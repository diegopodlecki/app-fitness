# Fitness App MVP - Guía de Inicio

¡Hola! Aquí tienes los pasos para probar tu nueva aplicación de fitness.

## Requisitos previos
Necesitas tener instalada la aplicación **Expo Go** en tu celular:
- [Android Play Store](https://play.google.com/store/apps/details?id=host.exp.exponent)
- [iOS App Store](https://apps.apple.com/us/app/expo-go/id982107779)

## Cómo ejecutar la app

1.  Abre una terminal en la carpeta del proyecto:
    `d:\miWeb1\fitness-app`

2.  Ejecuta el siguiente comando para iniciar el servidor de desarrollo:
    ```bash
    npx expo start
    ```

3.  Verás un código QR en la terminal.
    - **Android**: Abre Expo Go y escanea el código QR.
    - **iOS**: Abre la aplicación de Cámara y escanea el código QR (o usa Expo Go si te lo permite).

## Estructura de la App

La aplicación tiene 5 pestañas principales:
- **Inicio**: Panel principal con resumen.
- **Entrenar**: Pantalla de sesión de entrenamiento.
- **Historial**: Lista de entrenamientos pasados.
- **Descanso**: Temporizador.
- **Ejercicios**: Biblioteca con buscador y filtros.
